# JupyterLite HTML Demo

JupyterLite deployed as a [static site](https://vivek-component.github.io/jupyter) to GitHub Pages, for demo purposes.

## Requirements

JupyterLite is being tested against modern web browsers:

- Firefox 90+
- Chromium 89+
